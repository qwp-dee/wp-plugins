Sd Metabox Repetor Field is a simple WordPress plugin to Dynamically add/remove Input Fields in WordPress Metabox using Jquery on your wordpress page on your website.

== Description ==

Sd Metabox Repetor Fileds are a simple wordpress plugin which help you to add/remove your custom Inputs fileds on your wordpress pages.

This plugin is implemented in English as a default language but it is translatable so you can easily localize it.

Plugin Settings:

* Wordpress Page: It is Automatic initialize wordpress page.


Features:

* Easy to install.
* Easy to understand coding standard.
* You can add languages to localize it.
* Adding add remove fileds on your wordpress pages using custom metabox.


Plugin Develop by August Infotech [Visit website](http://www.augustinfotech.com)

== Installation ==

1. Upload the `sd-matabox-repetor-fileds` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin using the 'Plugins' inside wordpress pages.
